Application TyrellServices :

Environnrement conseillé : Windows
Etapes d'installation :
  - Télécharger et installer NodeJs
  - Ouvrir l'invite de commande et tapez les commandes suivantes :
    - npm install -g ionic cordova
    - ionic start TyrellServices --v2
    - Déplacez-vous dans le dossier TyrellServices ainsi créé
    - Vous pouvez d'ores et déjà lancer l'application avec la commande
      ionic serve
      A noté que vous pouvez rajouter l'option --lab pour visualiser l'application
      sur iOS, Android et Windows phone
